----------------------"INSTALLATION"--------------------
Before you start, you need to be sure about you have several tools as:

ddt
pytest
selenium
openpyxl
pytest-html
---------------------------"RUN"------------------------
In order for all test cases to be successful,
first run test cases test_Register and test_Address once.
Then you can run any test cases you want.
At the end of each test cases, the system will give you a report titled executed date.
